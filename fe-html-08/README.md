Сверстать [макет Zazor](/fe-html-08/homework6.psd):

1. Верстка шапки макета должна быть с пиксельной точностью
2. Обязательно использование тегов HTML5
3. Так же не забывайте о новых атрибутах форм, в хедере нужно применить `placeholder=""`

5. html-сруктуру, названия классов и стилизацию делаем с помощью методологии БЭМ
6. Три формы после баннера можно не верстать, это будет бонусным заданием, для тех кто хочет помучиться. Не обязательно, но рекомендовано к выполнению
